## 这个模组解决什么痛点

打蛋糕塔 Chapter 3 的玩家应该都有这种体验：

- Boss 栏顶上密密麻麻几行小字写着血量，**4 个 Boss 栏挤在一起根本看不清**
- 想知道自己有多少蓝心黑心，得手动 `/trigger ViewStats` 调出 50 多行文字，**刷屏遮挡视野**
- 想知道队友剩多少血，只能靠喊"我血多少多少"
- 群怪混战，**根本分不清自己正在打哪只**
- 打完一关只剩"我感觉我打得挺多"，**没法知道自己到底打了多少伤害**

这个模组把这些全部干掉。

## 详细功能介绍

### 一、屏幕底部 · 全新的状态条

> **[配图 3]** 屏幕底部状态条特写  
> 内容：彩色血条（建议拍**满血绿色 → 半血黄色 → 残血红色闪烁**3 个状态对比）+ 红/魂/黑/蓝心叠加 + 右下法力金币命数动量分栏  
> 推荐尺寸 1920×400（横长条比例）

- **彩色血条**：底下自带渐变血条，掉到一半变黄、四分之一变红还会闪 —— 危险时一眼就知道。
- **多色心叠加显示**：红心、魂心、黑心、蓝心 4 种心数实时刷新堆在一起，再也不用打 `ViewStats` 看一长串文字。
- **法力 / 金币 / 命数 / 动量**全部独立成条放在右下，像正经游戏 UI 而不是命令方块拼出来的字。

### 二、职业自适应（自动认你玩什么）

> **[配图 4]** 职业自适应对比图  
> 内容：左右分屏，**左 Joey 吸血鬼**（深红鲜血法力条）vs **右 Swan 果冻**（粉色冲刺动量图标）  
> 推荐尺寸 1920×800（左右拼图）

- 玩 **Joey 吸血鬼** 自动把法力换成深红鲜血条
- 玩 **Swan 果冻** 自动换成粉色冲刺动量图标
- 你切职业它就跟着变，**不用任何手动配置**

塔里那种把 Boss 栏文字替换成乱七八糟装饰的瞬间，下面这些条不会跟着 0 / 正常来回闪。

### 四、队友 · 一目了然

- **可拖拽**：能横排也能竖排。
- **队友头顶 3D 血条**：透墙可见，再也不用喊"我血多少多少快来奶我"。

> **[配图 7（可选）]** 队友头顶 3D 血条 in-game 实拍 + 透墙效果

- 自己 / 队伍 / 怪物三种 Boss 栏分开开关
- **不和别的 Boss 栏 mod 打架**

### 八、自动化 · 替你按命令

- 每 30 秒自动按一次 `ViewStats`，受伤或大额回血还会立刻刷新
- 这些刷新带来的"Your Stats:"提示会被**自动吃掉**，不刷你聊天栏
- 队伍 Boss 栏掉了会自动帮你 `TogglePartyBossbar`；带熔断保护，**不会因为狂按命令被服务器踢**

## 战绩统计（v6 新增 · 重磅）

> 不管你是单机存档还是和朋友联机的小服务器，只要**服务端装了这个 mod**就直接工作 —— 普通服务器也通过实测。

### 谁打了多少伤害，模组帮你算清楚

> **[配图 10]** 队友面板嵌入式战绩展示  
> 内容：把队友面板每行下方的小字段放大特写，能看清楚 `⚔ 344 ⛨ 155 ☠ 2 🤝 0 · 89/s`  
> 推荐尺寸 1200×600

塔里 9 种伤害类型（**Melee / Bullet / Force / Fire / Water / Ice / Dark / Light / Electric**）全部分开统计，**而且能正确分给真正开火的那个玩家**。

哪怕你和队友站一起、用召唤物在打，模组也会用一套优先级判断 —— 不会出现"伤害全算到一个人头上"的情况。

### 击杀 / 助攻 / 承伤

- **击杀**：致死那一下是谁打的就算谁，被免疫 / 救回 / 管理员一键清场都不会乱算。
- **助攻**：30 秒内打过它的其他人都给计上。
- **承伤**：你被怎么打掉的血也精确分玩家记录，可选在聊天栏播报"队友 A 这关吃了多少伤"。

### 嵌入到队友面板的实时战绩

队友面板每个人下面会多一行小字，长这样：

```
══════ T1F4 · 摩天大楼 · 00:46 ══════
 SimonBasil ⚔ 344 ⛨ 155 ☠ 2 🤝 0
 Kirin0321  ⚔ 180 ⛨  27 ☠ 1 🤝 0 · 89/s
 [全队]    ⚔ 524 ⛨ 182 ☠ 3 🤝 0
══════════ 全局 05:46 ══════════
 SimonBasil ⚔ 1220 ⛨ 540 ☠ 8 🤝 1
 Kirin0321  ⚔  890 ⛨ 220 ☠ 4 🤝 2
 [全队]    ⚔ 2110 ⛨ 760 ☠ 12 🤝 3
```

- ⚔ 造成的伤害
- ⛨ 承受的伤害
- ☠ 击杀
- 🤝 助攻
- `89/s` 是**最近 5 秒 DPS**

四种显示模式可选：

1. 不显示
2. 只显示当前这关
3. 只显示整局累计
4. 两者都显示（推荐）

没数据时自动隐藏，不挡视野。

## 安装方式

### 前置依赖

1. **Minecraft 1.21.4**
2. **Fabric Loader**（≥ 0.16.9）—— [https://fabricmc.net/use/installer/](https://fabricmc.net/use/installer/)
3. **Fabric API**（任意 1.21.4 兼容版本）—— [https://modrinth.com/mod/fabric-api](https://modrinth.com/mod/fabric-api)
4. **Java 21**（启动器自动下载，无需手动）
5. （可选但推荐）**ModMenu** —— [https://modrinth.com/mod/modmenu](https://modrinth.com/mod/modmenu) —— 用于配置本模组

### 安装本模组

1. 下载最新版 `cake-tower-hud-plus-X.X.X.jar`（**只下 jar 不带 sources 后缀的那个**）
2. 把 jar 放进 `.minecraft/mods/` 文件夹
3. 启动 Minecraft，进入游戏

> **[配图 14（可选）]** 截图展示 mods 文件夹结构，让新手看清楚 jar 应该放在哪里

### 服务器装吗？

- **单机 / 局域网开服 / 普通服务器**：服务端也装上同一个 jar 就能解锁 v6 战绩统计功能。
- **只装客户端**：HUD 部分照常工作，战绩统计部分会回落到本地缓存（功能受限）。
- **进入蛋糕塔地图自动启用，离开自动隐身**：不会在原版生存或其它服务器多出任何东西。

## 常见问题（FAQ）

### Q1. 这个 mod 会影响其它服务器吗？
**不会**。离开蛋糕塔地图时模组**自动隐身**，不会在原版生存或其它服务器多出任何东西。

### Q2. 需要 OP 权限或修改地图吗？
**都不需要**。所有命令都走地图自带的 `/trigger`，纯客户端 + 服务端 mod，不改地图、不改存档。

### Q3. 朋友联机要不要每个人都装？
- HUD 想用就装客户端。
- 战绩统计要工作必须**服务端**也装（房主 / LAN 开主机的人装就行）。
- 客户端单装：HUD 全部正常，战绩部分功能受限。

### Q4. 和 BetterF3 / WTHIT / Mod Menu 这些会冲突吗？
没有发现冲突。Boss 栏部分用了非破坏性拦截，与多数 Boss 栏类 mod 也能共存（仍建议实测）。

### Q5. 战绩统计准确吗？9 种伤害怎么算？
模组针对蛋糕塔的 9 种元素伤害类型设计了**8 层归属优先级**，能正确处理"召唤物代打""持续伤害（DoT）""玩家站在一起多人开火"等复杂场景；地图触发的剧情假伤害（如初始化跳血）会被识别成 L9-FILTER 子层、**不计入玩家账户**，避免乱算。

### Q6. 服务器卡顿怎么办？
v6.6.7 版本之后已经把高频伤害事件日志降级到 DEBUG，正常游玩 IO 开销已接近为零。仍卡的话欢迎反馈日志。

### Q7. 有云存档 / 跨设备同步吗？
战绩数据保存在世界存档下的 `data/ctt_stats.dat`，和地图存档一起备份就行。

### Q8. 地图作者会因为这个 mod 不开心吗？
**不会**。本 mod 只读地图的计分板和 Boss 栏文本，**不修改任何地图行为**；显示风格也是客户端独立绘制，不会破坏地图作者设计的"癫狂"特效（实际上模组还专门做了抗刷屏处理来保护地图体验）。
