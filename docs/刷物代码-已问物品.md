# 刷物代码（第二步计分板）
第一步先执行：`function cake_team_tower:misc/spawn_item`  
再执行下面对应的 `scoreboard` 行（对最近盔甲架写入池子与 ID）。  
天使池物品类型为 `RandomA...`，恶魔池为 `RandomD...`（与 `指令参考.md` 一致）。
